# My Next.js App

This is a [Next.js](https://nextjs.org) project featuring information about Auto, an AI coding assistant.

## Getting Started

First, run the development server:

```bash
npm run dev
# or
yarn dev
# or
pnpm dev
# or
bun dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

You can start editing the page by modifying `app/page.tsx`. The page auto-updates as you edit the file.

This project uses [`next/font`](https://nextjs.org/docs/app/building-your-application/optimizing/fonts) to automatically optimize and load [Geist](https://vercel.com/font), a new font family for Vercel.

## Deployment Guide

### Step 1: Add to GitHub/GitLab

1. **Create a new repository** on GitHub or GitLab:
   - Go to [GitHub](https://github.com/new) or [GitLab](https://gitlab.com/projects/new)
   - Create a new repository (e.g., `my-next-app`)
   - **Don't** initialize it with a README, .gitignore, or license

2. **Push your code** to the repository:
   ```bash
   git add .
   git commit -m "Initial commit with Auto AI assistant info"
   git branch -M main
   git remote add origin https://github.com/YOUR_USERNAME/my-next-app.git
   # or for GitLab:
   # git remote add origin https://gitlab.com/YOUR_USERNAME/my-next-app.git
   git push -u origin main
   ```

### Step 2: Sign Up to Vercel

1. Go to [Vercel.com](https://vercel.com)
2. Click **"Sign Up"**
3. Choose **"Continue with GitHub"** (or GitLab if you used GitLab)
4. Authorize Vercel to access your repositories

### Step 3: Connect and Deploy

1. After signing in, click **"Add New..."** â†’ **"Project"**
2. Import your `my-next-app` repository
3. Vercel will automatically detect it's a Next.js project
4. Click **"Deploy"** (you can use default settings)
5. Wait for the deployment to complete (usually 1-2 minutes)

### Step 4: Your Live Website

Once deployed, Vercel will provide you with a URL like:
- `https://my-next-app.vercel.app`
